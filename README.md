# Generator FirefoxOS

A yeoman generator for Firefox OS apps.

## Getting started

- Make sure you have [yo](https://github.com/yeoman/yo) installed:
    `npm install -g yo`
- Install the generator **locally**: `npm install generator-firefoxos`
- Run: `yo firefoxos`
